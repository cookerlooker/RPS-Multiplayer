Rock Paper Scissors Pseudocode 1. Each user presses r, p, or s on keyboard representing rock, paper, or scissors. 2. The user will not know what the other has chosen in a random value from a list of r, p, or s. 3. One of the users will win or not in accorndance to how the game is played. If user 1 picks rock and user 2 picks scissors then user 1 wins. If user 1 picks rock and user 2 picks paper then user 1 loses. If user 1 picks scissors and user 2 picks rock then user 1 loses. If user 1 picks scissors and user 2 picks paper then user 1 wins. If user 1 picks paper and user 2 picks rock then user 1 wins. If user 1 picks paper and user 2 picks scissors then user 1 loses. If user 1 picks the same as user 2 then they tie. 4. We then add to their score(s). If user 1 wins then we add one to their wins. If user 2 wins then we add one to their wins. If user 1 loses then we add one to their losses. If user 2 loses then we add one to their losses. If user2 ties then we add one to their ties. 5. The users then presses r, p, or s to play again.
